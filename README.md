# Java-Create-Word-Examples

This repository contains examples that illustrates how to create a new Word document with text, images, and tables in Java using Syncfusion’s [Word library](https://www.syncfusion.com/word-framework/net/word-library?utm_source=github&utm_medium=listing&utm_campaign=java-create-word-examples) (Essential DocIO) without Microsoft Word or Office interop dependencies.
